## Overview

The Aviation Data API provides comprehensive data for building flight search tools, travel apps, and aviation dashboards.

**Base URL**
```
https://aviation-api.logostream.com
```

## Quick start

```bash
curl "https://aviation-api.logostream.com/v1/airlines?iata=LH" \
  -H "x-api-key: YOUR_API_KEY"
```
