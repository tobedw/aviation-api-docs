---
title: Airlines
description: Retrieve airline profiles including colors, alliances, ratings, hubs, and editorial content.
---

## Endpoint

```
GET /v1/airlines
```

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `iata` | string | IATA code(s), comma-separated: `LH` or `LH,BA,AF` |
| `icao` | string | ICAO code: `DLH` |
| `name` | string | Fuzzy name search: `Emirates` |
| `alliance` | string | Alliance filter: `star_alliance`, `oneworld`, `skyteam` |
| `country_code` | string | ISO 2-letter country code: `DE` |
| `limit` | integer | Results per page (default: `50`) |
| `offset` | integer | Pagination offset (default: `0`) |

<Note>
  Querying without any filter (bulk download) requires a Startup plan or above.
</Note>

## Examples

<CodeGroup>

```bash Single airline
curl "https://aviation-api.logostream.dev/v1/airlines?iata=LH" \
  -H "x-api-key: YOUR_API_KEY"
```

```bash Multiple airlines
curl "https://aviation-api.logostream.dev/v1/airlines?iata=LH,BA,AF,DL,AA" \
  -H "x-api-key: YOUR_API_KEY"
```

```bash Search by name
curl "https://aviation-api.logostream.dev/v1/airlines?name=Emirates" \
  -H "x-api-key: YOUR_API_KEY"
```

```bash By alliance
curl "https://aviation-api.logostream.dev/v1/airlines?alliance=star_alliance" \
  -H "x-api-key: YOUR_API_KEY"
```

```bash By country
curl "https://aviation-api.logostream.dev/v1/airlines?country_code=DE" \
  -H "x-api-key: YOUR_API_KEY"
```

</CodeGroup>

## Response fields

| Field | Type | Description |
|---|---|---|
| `iata` | string | IATA 2-letter code |
| `icao` | string | ICAO 3-letter code |
| `three_digit_code` | string | IATA 3-digit AWB prefix (e.g. `020` for Lufthansa) |
| `callsign` | string | ICAO radio callsign (e.g. `LUFTHANSA`) |
| `name` | string | Official airline name |
| `type` | string | Carrier type: `full_service`, `low_cost`, `regional`, `charter`, `cargo`, `helicopter` |
| `category` | string | `Passenger`, `Cargo`, or `null` |
| `alliance` | string | `star_alliance`, `oneworld`, `skyteam` or `null` |
| `airline_group` | string | Parent group (e.g. `Lufthansa Group`) |
| `country_code` | string | ISO 2-letter country code |
| `home_region` | string | Geographic region |
| `hq_address` | string | Headquarters address |
| `colors` | string | Primary brand color (hex) |
| `color2–color6` | string | Additional brand colors (hex) |
| `text_color` | string | Text color on brand background (hex) |
| `text_color2` | string | Secondary text color (hex) |
| `description` | string | Short airline description |
| `long_description` | string | Extended editorial description |
| `is_oneworld` | boolean | oneworld member |
| `is_staralliance` | boolean | Star Alliance member |
| `is_skyteam` | boolean | SkyTeam member |
| `year_founded` | integer | Year the airline was founded |
| `commenced_operations` | string | Year operations commenced |
| `active` | boolean | Whether the airline is currently operating |
| `defunct` | string | Year the airline ceased operations (e.g. `"2021"`), or `null` |
| `defunct_reason` | string | Reason for ceasing operations |
| `rating_tripadvisor` | number | TripAdvisor rating (1–5) |
| `rating_skytrax_stars` | number | Skytrax star rating |
| `website` | string | Official website URL |
| `phone` | string | Contact phone number |
| `wikipedia_url` | string | Wikipedia article URL |
| `instagram_url` | string | Instagram profile URL |
| `twitter_url` | string | Twitter / X profile URL |
| `facebook_url` | string | Facebook page URL |
| `youtube_url` | string | YouTube channel URL |
| `linkedin_url` | string | LinkedIn company URL |
| `tiktok_url` | string | TikTok profile URL |
| `hubs` | array | Curated primary/secondary hub airports. Each entry: `{airport_iata, notes}`. |
| `focus_cities` | array | Curated focus cities. Same shape as `hubs`. |
| `fleet_size` | integer | Number of aircraft in the fleet |
| `logo` | object | Logo paths for the Logo API (see below) |

### Logo object

| Field | URL |
|---|---|
| `icon` | `https://airlines-api.logostream.dev/airlines/icao/{ICAO}?key=…` |
| `logo` | `https://airlines-api.logostream.dev/airlines/icao/{ICAO}?variant=logo&key=…` |
| `logo_white` | `https://airlines-api.logostream.dev/airlines/icao/{ICAO}?variant=logo-white&key=…` |
| `tail` | `https://airlines-api.logostream.dev/airlines/icao/{ICAO}?variant=tail&key=…` |

The `logo` object contains fully qualified URLs including your API key. Use them directly:

```javascript
// Ready to use — no modification needed
<img src={airline.logo.icon} />
<img src={airline.logo.logo} />
```
