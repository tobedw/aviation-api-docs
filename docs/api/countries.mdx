---
title: Countries
description: Retrieve country data with ISO codes and continent information.
---

## Endpoint

```
GET /v1/countries
```

The full list of 252 countries is available on all plans without requiring a filter.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `code` | string | ISO 2-letter country code: `DE` |
| `name` | string | Fuzzy name search: `germany` |
| `continent` | string | Continent code: `EU` |
| `limit` | integer | Results per page (default: `50`) |
| `offset` | integer | Pagination offset (default: `0`) |

**Continent codes:** `AF` Africa Â· `AN` Antarctica Â· `AS` Asia Â· `EU` Europe Â· `NA` North America Â· `OC` Oceania Â· `SA` South America
