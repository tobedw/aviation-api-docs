## Why use it

Everything your destination UI needs.

<CardGroup cols={2}>
  <Card title="Popularity-Sorted Results" icon="arrow-up-wide-short">
    Cities ranked by a composite travel popularity score — great destinations bubble up automatically.
  </Card>
  <Card title="Fuzzy Name Matching" icon="magnifying-glass">
    Prefix and contains search across all city names. Power autocomplete with a single API call.
  </Card>
  <Card title="Full Geo Context" icon="location-dot">
    Coordinates, timezone, state, and country — everything needed to render a destination card.
  </Card>
  <Card title="Localized Names" icon="language">
    Pass `lang=` for localized city and country names. Supports all major BCP 47 language tags.
  </Card>
</CardGroup>

## Endpoint

```
GET /v1/cities
```

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `iata` | string | City IATA code: `LON` |
| `name` | string | Fuzzy city name search: `london` |
| `country` | string | ISO 2-letter country code: `GB` |
| `limit` | integer | Results per page (default: `50`) |
| `offset` | integer | Pagination offset (default: `0`) |

<Note>
  Querying without any filter (bulk download) requires a Startup plan or above.
</Note>

## Examples

<CodeGroup>

```bash By IATA code
curl "https://aviation-api.logostream.dev/v1/cities?iata=LON" \
  -H "x-api-key: YOUR_API_KEY"
```

```bash Search by name
curl "https://aviation-api.logostream.dev/v1/cities?name=london" \
  -H "x-api-key: YOUR_API_KEY"
```

```bash All cities in a country
curl "https://aviation-api.logostream.dev/v1/cities?country=GB" \
  -H "x-api-key: YOUR_API_KEY"
```

</CodeGroup>

## Response fields

| Field | Type | Description |
|---|---|---|
| `iata_code` | string | City IATA code |
| `city_name` | string | City name |
| `state_short` | string | State/province abbreviation |
| `state_full` | string | State/province full name |
| `latitude` | number | Decimal latitude |
| `longitude` | number | Decimal longitude |
| `timezone` | string | IANA timezone |
| `population` | integer | City population |
| `popularity_score` | integer | Relative travel popularity score (0–100) |
| `wikipedia_url` | string | Wikipedia article URL |
| `website` | string | Official city website |
| `country` | object | `{code2, name, continent_code}` |
