title: Airports
description: Retrieve airport profiles with traffic stats, city and country data.
---

## Endpoint

```
GET /v1/airports
```

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `iata` | string | IATA code(s), comma-separated: `LHR` or `LHR,CDG,JFK` |
| `icao` | string | ICAO code: `EGLL` |
| `city_name` | string | Fuzzy city name search: `London` |
| `country_code` | string | ISO 2-letter country code: `DE` |
| `major_only` | integer | Set to `1` to return only large airports |
| `limit` | integer | Results per page (default: `50`) |
| `offset` | integer | Pagination offset (default: `0`) |

Results are sorted by annual passenger count (busiest first) when using filters or bulk queries.

<Note>
  Querying without any filter (bulk download) requires a Startup plan or above.
</Note>

## Examples

<CodeGroup>

```bash Single airport
curl "https://aviation-api.logostream.dev/v1/airports?iata=LHR" \
  -H "x-api-key: YOUR_API_KEY"
```

```bash Multiple airports
curl "https://aviation-api.logostream.dev/v1/airports?iata=LHR,CDG,JFK,NRT,SIN" \
  -H "x-api-key: YOUR_API_KEY"
```

```bash Airports by city
curl "https://aviation-api.logostream.dev/v1/airports?city_name=London" \
  -H "x-api-key: YOUR_API_KEY"
```

```bash Major airports in Germany
curl "https://aviation-api.logostream.dev/v1/airports?country_code=DE&major_only=1" \
  -H "x-api-key: YOUR_API_KEY"
```

</CodeGroup>

## Response fields

| Field | Type | Description |
|---|---|---|
| `iata` | string | IATA 3-letter code |
| `icao` | string | ICAO 4-letter code |
| `name` | string | Airport name |
| `type` | string | `large_airport`, `medium_airport`, `small_airport`, `heliport`, `seaplane_base` |
| `latitude` | number | Decimal latitude |
| `longitude` | number | Decimal longitude |
| `elevation_ft` | integer | Elevation above sea level in feet |
| `timezone` | string | IANA timezone (e.g. `Europe/London`) |
| `passengers` | integer | Annual passenger count |
| `aircraft_movements` | integer | Annual aircraft movements |
| `cargo_tonnes` | integer | Annual cargo volume in tonnes |
| `hubs` | array | Airlines using this airport as a hub or focus city. Each entry: `{airline_iata, airline_icao, airline_name, type, notes}` where `type` is `hub` or `focus_city`. Sourced from the curated [airline_hubs](/docs/api/hubs) table. |
| `terminals` | string | Terminal count or names |
| `website` | string | Official airport website |
| `phone` | string | Contact phone number |
| `address` | string | Physical address |
| `description` | string | Airport description |
| `wikipedia_url` | string | Wikipedia article URL |
| `active` | boolean | Whether the airport is currently active |
| `city` | object | `{iata_code, city_name, latitude, longitude}` |
| `country` | object | `{code2, name, continent_code}` |
| `logo` | object | Logo paths for the Logo API (see below) |
