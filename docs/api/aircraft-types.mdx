---
title: Aircraft Types
description: Retrieve aircraft type names by IATA code.
---

## Endpoint

```
GET /v1/aircraft-types
```

The full list of 74 aircraft types is available on all plans.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `iata` | string | IATA code(s), comma-separated: `32S` or `32S,787,350` |
| `name` | string | Fuzzy name search: `Boeing` |
| `limit` | integer | Results per page (default: `50`) |
| `offset` | integer | Pagination offset (default: `0`) |

## Examples

<CodeGroup>

```bash Single type
curl "https://aviation-api.logostream.dev/v1/aircraft-types?iata=32S" \
  -H "x-api-key: YOUR_API_KEY"
```

```bash Multiple types
curl "https://aviation-api.logostream.dev/v1/aircraft-types?iata=32S,787,350,380" \
  -H "x-api-key: YOUR_API_KEY"
```

```bash Search by name
curl "https://aviation-api.logostream.dev/v1/aircraft-types?name=Boeing" \
  -H "x-api-key: YOUR_API_KEY"
```

```bash Full list
curl "https://aviation-api.logostream.dev/v1/aircraft-types" \
  -H "x-api-key: YOUR_API_KEY"
```

</CodeGroup>

## Response fields

| Field | Type | Description |
|---|---|---|
| `iata` | string | IATA aircraft type code |
| `icao` | string | ICAO aircraft type designator |
| `name` | string | Aircraft name |
| `manufacturer` | string | Manufacturer name |
| `category` | string | e.g. `wide_body`, `narrow_body`, `regional_jet` |
| `first_flight` | date | Date of first flight |
| `typical_seats` | integer | Typical seat count in mixed configuration |
| `range_km` | integer | Maximum range in kilometres |
| `wikipedia_url` | string | Wikipedia article URL |
| `description` | string | Aircraft description |
| `airlines` | array | Airlines operating this type |

**Airline object within `airlines`:**

| Field | Type | Description |
|---|---|---|
| `count` | integer | Number of aircraft in the fleet |
| `airline` | object | `{iata, name}` |

## Common codes

| IATA | Aircraft |
|---|---|
| `32S` | Airbus A320 Family |
| `330` | Airbus A330 |
| `350` | Airbus A350 |
| `380` | Airbus A380 |
| `737` | Boeing 737 |
| `747` | Boeing 747 |
| `777` | Boeing 777 |
| `787` | Boeing 787 Dreamliner |
| `CRJ` | Bombardier CRJ |
| `EMJ` | Embraer E-Jet |
