---
title: Routes
description: Retrieve flight routes with airline, frequency, duration, and cabin class data.
---

## Endpoint

```
GET /v1/routes
```

<Warning>
  At least one filter parameter is required.
</Warning>

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `departureIata` | string | Departure airport IATA: `FRA` |
| `departureIcao` | string | Departure airport ICAO: `EDDF` |
| `arrivalIata` | string | Arrival airport IATA: `LHR` |
| `airlineIata` | string | Airline IATA code: `LH` |
| `airlineIcao` | string | Airline ICAO code: `DLH` |
| `limit` | integer | Results per page (default: `50`) |
| `offset` | integer | Pagination offset (default: `0`) |

Parameters can be combined, e.g. `departureIata=FRA&arrivalIata=JFK` to look up a specific route.

## Examples

<CodeGroup>

```bash All routes from Frankfurt
curl "https://aviation-api.logostream.dev/v1/routes?departureIata=FRA" \
  -H "x-api-key: YOUR_API_KEY"
```

```bash Specific route
curl "https://aviation-api.logostream.dev/v1/routes?departureIata=FRA&arrivalIata=JFK" \
  -H "x-api-key: YOUR_API_KEY"
```

```bash All Lufthansa routes
curl "https://aviation-api.logostream.dev/v1/routes?airlineIata=LH" \
  -H "x-api-key: YOUR_API_KEY"
```

```bash Filter by airline ICAO
curl "https://aviation-api.logostream.dev/v1/routes?airlineIcao=DLH" \
  -H "x-api-key: YOUR_API_KEY"
```

```bash All routes to London Heathrow
curl "https://aviation-api.logostream.dev/v1/routes?arrivalIata=LHR" \
  -H "x-api-key: YOUR_API_KEY"
```

</CodeGroup>

## Response

```json
{
  "data": [
    {
      "id": 12345,
      "departure_iata": "FRA",
      "arrival_iata": "JFK",
      "airlines": [
        { "airline_iata": "LH", "airline_icao": "DLH" },
        { "airline_iata": "UA", "airline_icao": "UAL" }
      ]
    }
  ],
  "total": 1,
  "truncated": false,
  "page": 1,
  "limit": 50
}
```

## Response fields

| Field | Type | Description |
|---|---|---|
| `id` | integer | Route ID |
| `departure_iata` | string | Departure airport IATA |
| `arrival_iata` | string | Arrival airport IATA |
| `airlines` | array | Airlines operating this route |

**Airline object within `airlines`:**

| Field | Type | Description |
|---|---|---|
| `airline_iata` | string | Airline IATA code |
| `airline_icao` | string | Airline ICAO code |
