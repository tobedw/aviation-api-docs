---
title: Routes
description: Retrieve flight routes with airline, frequency, duration, and cabin class data.
---

## Endpoint

```
GET /v1/routes
```

<Warning>
  At least one filter parameter is required.
</Warning>

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `departureIata` | string | Departure airport IATA: `FRA` |
| `departureIcao` | string | Departure airport ICAO: `EDDF` |
| `arrivalIata` | string | Arrival airport IATA: `LHR` |
| `airlineIata` | string | Airline IATA code: `LH` |
| `limit` | integer | Results per page (default: `50`) |
| `offset` | integer | Pagination offset (default: `0`) |

Parameters can be combined, e.g. `departureIata=FRA&arrivalIata=JFK` to look up a specific route.

## Examples

<CodeGroup>

```bash All routes from Frankfurt
curl "https://aviation-api.logostream.com/v1/routes?departureIata=FRA" \
  -H "x-api-key: YOUR_API_KEY"
```

```bash Specific route
curl "https://aviation-api.logostream.com/v1/routes?departureIata=FRA&arrivalIata=JFK" \
  -H "x-api-key: YOUR_API_KEY"
```

```bash All Lufthansa routes
curl "https://aviation-api.logostream.com/v1/routes?airlineIata=LH" \
  -H "x-api-key: YOUR_API_KEY"
```

```bash All routes to London Heathrow
curl "https://aviation-api.logostream.com/v1/routes?arrivalIata=LHR" \
  -H "x-api-key: YOUR_API_KEY"
```

</CodeGroup>

## Response fields

| Field | Type | Description |
|---|---|---|
| `departure_iata` | string | Departure airport IATA |
| `departure_icao` | string | Departure airport ICAO |

| `arrival_icao` | string | Arrival airport ICAO |
| `duration_min` | integer | Flight duration in minutes |
| `flights_per_week` | integer | Weekly frequency |
| `flights_per_day` | string | Daily frequency range |
| `price` | integer | Approximate price in USD |

| Field | Type | Description |
|---|---|---|
| `airline_iata` | string | Airline IATA code |
| `airline_name` | string | Airline name |
| `aircraft_codes` | string | Comma-separated aircraft IATA codes (e.g. `"32S,787"`) |
| `class_first` | boolean | First class available |
| `class_business` | boolean | Business class available |
| `class_economy` | boolean | Economy class available |
| `lcc` | boolean | Low-cost carrier |
| `is_new` | boolean | Newly added route |
| `first_flight` | date | Date this airline started the route |
