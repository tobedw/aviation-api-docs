---

All tools are called via `tools/call` with a `name` and `arguments` object.

Parameters marked `*` are required. For tools that say "provide one of", at least one of the listed parameters must be present.

---

## `get_airline`

Look up an airline by IATA or ICAO code.

| Parameter | Type | Required | Description |
|---|---|---|---|
| `iata` | string | one of | 2-letter IATA code, e.g. `LH` |
| `icao` | string | one of | 3-letter ICAO code, e.g. `DLH` |

```json
{ "name": "get_airline", "arguments": { "iata": "LH" } }
```

---

## `get_airport`

Look up an airport by IATA or ICAO code.

| Parameter | Type | Required | Description |
|---|---|---|---|
| `iata` | string | one of | 3-letter IATA code, e.g. `FRA` |
| `icao` | string | one of | 4-letter ICAO code, e.g. `EDDF` |

```json
{ "name": "get_airport", "arguments": { "iata": "FRA" } }
```

---

## `get_routes`

List routes filtered by origin, destination, and/or airline. Provide at least one filter.

| Parameter | Type | Required | Description |
|---|---|---|---|
| `from` | string | one of | Origin airport IATA |
| `to` | string | one of | Destination airport IATA |
| `airline` | string | one of | Airline IATA |
| `limit` | integer | — | Max results, 1–50 (default 20) |
| `offset` | integer | — | Pagination offset (default 0) |

```json
{ "name": "get_routes", "arguments": { "from": "FRA", "to": "JFK" } }
```

---

## `get_route_map`

Generate a great-circle route map as SVG or PNG.

| Parameter | Type | Required | Description |
|---|---|---|---|
| `stops` | string | one of | Comma-separated IATA codes, e.g. `FRA,DXB,SYD` |
| `from` | string | one of | Origin airport IATA |
| `to` | string | one of | Destination airport IATA |
| `format` | string | — | `svg` or `png` (default `svg`) |

Returns raw SVG/PNG content as a string in `result.content[0].text`.

```json
{ "name": "get_route_map", "arguments": { "stops": "FRA,DXB,SYD", "format": "svg" } }
```

## `get_stats`

Network statistics for an airport or airline. Provide exactly one of `airport` or `airline`.

| Parameter | Type | Required | Description |
|---|---|---|---|
| `airport` | string | one of | Airport IATA, e.g. `FRA` — exactly 3 uppercase letters |
| `airline` | string | one of | Airline IATA, e.g. `LH` — exactly 2 uppercase letters |

```json
{ "name": "get_stats", "arguments": { "airport": "FRA" } }
```

## Pagination

Paginated tools (`get_routes`, `get_routes_country_pair`, `get_hubs`) support:

| Parameter | Default | Max |
|---|---|---|
| `limit` | 20 | 50 |
| `offset` | 0 | — |

```json
{ "name": "get_routes", "arguments": { "from": "FRA", "limit": 50, "offset": 50 } }
```

---

## Input Validation

| Field | Format | Error |
|---|---|---|
| `get_stats.airport` | `/^[A-Z]{3}$/` | `airport must be a 3-letter IATA code.` |
| `get_stats.airline` | `/^[A-Z]{2}$/` | `airline must be a 2-letter IATA code.` |
| `get_airport_diagram.icao` | `/^[A-Z]{4}$/` | `icao must be a 4-letter ICAO code.` |

All codes must be uppercase. Lowercase input will be rejected.
