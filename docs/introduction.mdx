Logostream provides two complementary APIs for aviation data:

<CardGroup cols={2}>
  <Card title="Aviation Data API" icon="database" href="/data/authentication">
    REST API for airlines, airports, cities, countries, routes, and aircraft types. Returns structured JSON data.

    **Base URL:** `https://aviation-api.logostream.dev`
  </Card>
  <Card title="Logo API" icon="image" href="/logo/authentication">
    Direct image delivery for airline logos, airport logos, and country flags. Returns SVG or PNG assets.

    **Base URL:** `https://airline-api.logostream.dev`
  </Card>
</CardGroup>

## Same API key

Both APIs use the same API key — one key gives you access to everything.

[Get your API key →](https://airline.logostream.dev)
